Blockspaces uses the OpenGL coordinate system, where +X goes from left to right, +Y goes from bottom to top, and +Z comes out of the screen.

```
                         _________________________ (1,1,0)
                        / _________Top_________  /|
                       / / ___________________/ / |   
                      / / /| |               / /  |  /|\
                     / / / | |              / / . |   |
                    / / /| | |             / / /| |   |
                   / / / | | |            / / / | |   |
                  / / /  | | |           / / /| | |   | +Y axis
                 / /_/__________________/ / / | | |   |
                /________________________/ /  | | |   |
        Left--> | ______________________ | |  | | |   | 
                | | |    | | |_________| | |__| | |   |
                | | |    | |___________| | |____| |   |
                | | |   / / ___________| | |_  / /    
                | | |  / / /           | | |/ / /     /
                | | | / / /            | | | / /     /
                | | |/ / /             | | |/ /     /
                | | | / /              | | ' /     /  +Z axis
                | | |/_/_______________| |  /     /
                | |____________________| | /     /
                |________Front___________|/    \/
             (0,0,1)
                  ---------------------->
                        +X axis
```
